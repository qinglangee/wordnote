
# word
this is a page used to learn english words



## Usage
1. open  config/serverConf.js, set base dir
2. 把`init.js` 中最后一句 `// initDatabase();` 的注释去掉, 然后执行 `node routes/base/init.js`
3. 可以 node app.js 正常启动了
4. 再重新把 `initDatabase();` 给注释掉．
